# 🎵 Top 50 Spotify Songs Visualization

## 📌 Project Overview
This project analyzes the Top 50 Spotify songs dataset using data visualization techniques.

📊 The dashboard helps identify:
- Most popular artists
- Song popularity trends
- Top streamed songs
- and more...

---

## 🛠 Tools Used
- Microsoft Power BI
- Microsoft Excel

---

## 📂 Dataset
Dataset contains:
- date
- position
- song
- artist
- popularity
- duration_ms
- album_type
- total_tracks
- release_date
- is_explicit
- album_cover_url
